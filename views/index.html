Timestamp Microservice

Project #1

Example Usage:

  • https://fcc-project-timestamp.julianbattaglin.repl.co/api/ Empty date parameter return the current time in a JSON object with a unix/utc key.
  • https://fcc-project-timestamp.julianbattaglin.repl.co/api/2015-12-25 A request to /api/:date? with a valid date return a JSON object with a utc key that is a string of the input date in the format: Thu, 01 Jan 1970 00:00:00 GMT
  • https://fcc-project-timestamp.julianbattaglin.repl.co/api/1451001600000 A request to /api/1451001600000 return { unix: 1451001600000, utc: "Fri, 25 Dec 2015 00:00:00 GMT" }

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
